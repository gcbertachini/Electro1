\section{Funcionamiento}
Para analizar la operación del circuito se puede pensar en los siguientes casos:

\begin{enumerate}
    \item $|V_i| \leq V_f$
    \item $V_f < |V_i| \leq V_z + V_f$
    \item $V_z + V_f < |V_i|$ 
\end{enumerate}
